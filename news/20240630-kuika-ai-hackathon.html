AI Hackathon Experience

June 30, 2024

As a member of the Finetuners team, I attended the AI Hackathon organized by Kuika and the Karşıyaka Municipality. Our team comprised Cüneyd Çelik and Gencay Turgut. We had an excellent time working together, with Gencay focusing on the front-end, Cüneyd connecting the backend and frontend, and myself creating agents.

The hackathon started on Saturday and ended on Sunday. During the event, I learned a lot about the OpenAI API, LangChain, and Multiagent LLM models. As a team, we utilized tools such as Flask, Python, LangChain, and React.

Although we didn't place in the top three, we gained a wealth of knowledge, made new friends, and created valuable connections with other participants. The mentors and jury were impressive, and it was a great networking opportunity.

Surprisingly, we won a team prize of 5000 TL and received delightful gifts from the Kuika team.

Project Overview: AI Assistant

This is the project we created during the Kuika AI Hackathon as the Finetuners team.

Agents

The project includes the following agents to handle specific types of queries:

  • Context Decider Agent: Decides which department (agent) should handle the user's query. Utilizes langchain.prompts and ChatOpenAI.
  • Code Analysis Agent: Analyzes Python code files and answers code-related questions. Uses the OpenAI API for code analysis.
  • Document Agent: Processes and analyzes various document types (PDF, DOCX, PPTX). Extracts and summarizes content from documents.
  • Database Agent: Executes SQL queries on a SQLite database and provides explanations for query results.
